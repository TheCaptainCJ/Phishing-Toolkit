# Phishing Analysis Toolkit

A Python-based tool that extracts URLs from email text, checks reputation using the URLhaus API, and generates a clean investigation report.

## Features
- Extracts URLs using regex
- Checks reputation via URLhaus (no API key needed)
- Generates readable investigation reports
- Modular code structure
- Includes unit tests

## How to Run
pip install -r requirements.txt
python main.py

## How to Test
pytest
